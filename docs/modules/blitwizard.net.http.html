Module blitwizard.net.http

This is the HTTP module which allows you to fetch web documents.

You can use this to automatically download patches/updates, high score lists etc.

Functions

get (url, response_callback)
Use this function to get a web document.

The function will issue a callback once the requested document is available.

Parameters:

  • url string the url of the web document, e.g. http://www.blitwizard.de/
  • response_callback function the function which will be called once the document arrived (or when arrival failed). The first and only paramater is a response table that contains the document (or the error on failure) and various additional information.

Usage:

       blitwizard.net.http.get("http://www.blitwizard.de/",
       function(response)
           if response.response_code == 200 then
               print("Document arrived! Here it is:\n" .. response.content)
           else
               print("Failed to obtain document! Error code: " ..
                   response.response_code)
           end
       end
   )

Tables

response
This illustrates the response table you get from the http.get function's reponse_callback.

Fields:

  • response_code number the HTTP response code. 200 means success, anything else indicates an error. All errors are usually indicated by the server, search the internet for HTTP response codes if you need more detailed information on how to interpret them. The exception is response code 999, which issued locally by blitwizard in case the server couldn't be contacted at all.
  • string headers contains a string with all the response HTTP headers sent by the server. This is for advanced HTTP tricks, and usually not needed.
